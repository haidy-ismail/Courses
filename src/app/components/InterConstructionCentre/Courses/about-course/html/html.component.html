    • عليك أن تعلم أن لغة html لغة وصفية، أي فقط تجهيز لصفحة الويب بدون أي نشاط وبدون أي جماليات للصفحة، يمكن أن تشبهها بهيكل الإنسان العظمي ثم لغة ال css لغة تجميلية لما بنيته بال html، أي لتضيف بعض الجماليات على صفحة الويب، الألوان، الخطوط، أشكال الأزرار وغيره الكثير وهي بحر واسع، بالإضافة إلى js، لتشكل منظومة كاملة تسمى بال front-end، لو تمكنت من احتراف ذلك كله، يمكن الاكتفاء بها، كمبرمج فرونت اند، مع حاجتك لاستخدام بعض ال framework التي ستريحك جداً مثل bootstrap 5 وهو يجمع كل من html css js في مكان واحد.


    لكى تبنى موقع يجب ان تتعلم HTML أولا ثم يتبعه CSS، وأنصح بتلك الطريقة لأن تطوير الويب يكون مفهوما جديدا بالنسبة لك، وأهم شيء في تلك المرحلة هو أن تتعلم "الإنشاء" والهيكلة وإضافة الصور والجداول والكائنات التفاعلية مثل النماذج أو ملفات الفيديو والصوت، وعرض المحتوى واستعمال مختلف أنواع الوسم في HTML.


    وبالمناسبة يمكنك بناء موقع بأكمله باستعمال HTML فقط ودون الحاجة لأي شيء آخر، طبعا لن يكون موقعا عمليا وفعالا بالنظر لما وصل إليه تطوير الويب اليوم، لكن ذلك مثال عن أهمية HTML في تعلم تطوير الويب.


    أما CSS فيأتي لاحقا ليكون مكملا جماليا يعني بعزل التنسيق (الألوان والخطوط والأزرار) فتضيفه لصفحة الويب التي صممتها باستعمال HTML لتعريف شكل وتخطيط المحتوى.
